# Issue Tracker

This repo uses GitHub Issues in `monteleecode/side-projects-horsefarm`.

Use `gh` for issue-aware workflows that need to create, update, or inspect issues.

For shipping code, use the `github:yeet` skill to:

- create or reuse the pull request
- include `Fixes #<issue>` or `Closes #<issue>` in the PR body
- merge the PR
- delete the source branch
- sync local `main`
